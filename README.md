# Be API Critical App
Electron Application to generate Critical CSS (WIP)

## Installation

Install dependencies using npm or yarn
```shell
$ npm install
```

```shell
$ yarn
```
Start create-react-app server with
```shell
$ npm start
```

Launch Electron app in another terminal tab with
```shell
$ npm run electron
```
